RunOneDaemon contains
  * scripts and code to execute one LOCKSS daemon
  * which receives its configuiration settings from the MetaArchive cooperatives conspectus instances
  * and which uses the plugins from this project's trunk to test harvest content
  * details are given in the [README](http://educopia.googlecode.com/svn/branches/release/plugins_dev/run_one_daemon/README) file

You can download it from /branches/release/plugins\_dev/run\_one\_daemon/
> svn checkout http://educopia.googlecode.com/svn/branches/release/plugins_dev/run_one_daemon/

Then set it up according to the instructions in the README file that comes with run\_one\_daemon and read through the instructions about how to use it given in the README file as well.
